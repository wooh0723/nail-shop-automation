UI 작업 완료 후 아래 기준을 스스로 평가하고 미달이면 재시도. 최대 10회.
- 네일 이미지가 충분히 돋보이는가
- B&W 톤이 일관되게 유지되는가
- 불필요한 요소가 없는가

기준 미달 항목이 하나라도 있으면 스스로 수정 후 재평가할 것

## 스크린샷 기반 자율 개선 루프
Puppeteer로 실제 화면을 보고 평가하는 루프를 실행할 것.

### 루프 순서
1. `npm run dev` 실행 (백그라운드)
2. `scripts/screenshot.ts` 로 localhost:3000 스크린샷 캡처
3. 캡처된 이미지를 직접 보고 아래 기준으로 평가
4. 미달 항목 수정 후 1번부터 반복
5. 모든 기준 통과 시 루프 종료, `npm run build` 로 최종 검증

### 평가 기준
- 네일 이미지가 충분히 돋보이는가
- B&W 톤이 일관되게 유지되는가 (악센트 컬러 없음)
- 불필요한 요소가 없는가
- 모바일(390px) 여백과 타이포가 균형잡혀 있는가
- 프리미엄 네일샵 룩북다운 감도가 느껴지는가
- 시기 드롭다운이 오른쪽 정렬로 정확히 동작하는가
- 가격 nudge 애니메이션이 자연스럽게 동작하는가

### 스크린샷 스크립트 생성
scripts/screenshot.ts 가 없으면 먼저 생성할 것:
- localhost:3000 전체 페이지 캡처
- 모바일 뷰포트 390x844 기준
- 저장 경로: scripts/screenshots/latest.png
- 최대 10회 반복
